Note App

Simple React note app (JSX) with Add / Update / Delete functionality.

Quick start

1. Install dependencies:

```
npm install
```

2. Run dev server:

```
npm run dev
```

Open the URL shown by Vite (usually http://localhost:5173).

Files created

- package.json
- index.html
- src/main.jsx
- src/App.jsx
- src/styles.css

This is intentionally minimal — a light design with bordered cards and basic controls.
